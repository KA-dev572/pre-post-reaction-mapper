# 発信前リアクション可視化ツール(pre-post-reaction-mapper)
<!--マークダウン形式の書式習得も兼ねる-->
## 本ツールの目的・設計思想（Purpose）
本ツールは、日本に本拠を置く企業がSNS（特にX）を運用する際に、<br>
**投稿者の意図とは無関係に、第三者から政治的・社会的な文脈を付与され、炎上に発展するリスクを低減する**ことを目的としています。<br>
企業アカウントによる投稿は、
- 善意
- ポジティブな意図
- 軽いユーモア
- 内輪的な感覚<br>

で行われたものであっても、<br>
**投稿時期・語彙・発信主体（企業であること）の組み合わせによって、意図しない意味づけや批判の対象となる**ことがあります。<br><br>
本ツールは、そうしたリスクが生じやすい語彙を機械的に抽出し、<br>
**「投稿前に一度立ち止まって検討するための材料」を提供する**補助ツールです。

### 想定ユーザ（Target Users）
- 日本に本拠を置く企業・団体
- 広報・マーケティング目的でSNS運用を行っている
- SNS運用の経験が浅い、または属人的な判断に依存している
- 専門の炎上対策チームや法務レビューを常時通せない規模<br>

※個人利用を妨げるものではありませんが、本ツールは企業アカウント運用を主な対象として設計されています。

### 想定シーン（Use Cases）
本ツールは、以下のような場面での利用を想定しています。<br>
投稿文に含まれる語彙が、
- 特定の日付・記念日・歴史的出来事と結びつき、政治的意図や価値判断を疑われる可能性がある場合
- 特定の属性・集団・価値観に対して揶揄・嘲笑・マウントを行っていると解釈される可能性がある語彙が含まれている場合
- 自己陶酔・精神論・過剰な献身の強要と受け取られる可能性がある場合
- 上記の表現が繰り返し使用される場合<br>

これらの語彙を自動的に検出し、別表現への言い換えや投稿タイミング再検討といった意思決定を行うための情報を提示します。

### 本ツールが「やらないこと」
本ツールは、以下のことを目的としていないため、設計にあえて組み込んでいません。
1. 表現の価値判断・正邪の断定<br>
本ツールは、
- 表現の善悪
- 政治的・歴史的評価
- 社会的に正しい／間違っているか<br>

  といった判断を**行いません**。<br>
  あくまで、<br>
  **文脈を共有していない第三者から、どのように誤解され得るか**<br>
  というリスクを示すのみです。<br>
  いわゆる「自主検閲」「言葉狩り」「啓蒙」を目的としたものでは**ありません**。

2. 文脈依存の高度な判断<br>
本ツールは、現時点では
- 単語の出現有無
- 投稿予定日との単純な対応
- 使用頻度<br>

  といった機械的に判定可能な要素のみを扱います。<br>
  文脈理解、皮肉、引用、歴史的評価の違いなどを考慮した判断は行いません。

3. 修正案・代替表現の提示<br>
本ツールは、文脈理解を伴わない状態での自動生成による言い換えを行いません。<br>
修正の要否や表現の選択は、最終的にユーザー自身の判断と責任に委ねられます。

### 本ツールの位置づけ
本ツールは、<br>
**「投稿してはいけない表現を決めるツール」ではなく**、<br>
**「投稿前に考えるためのチェックリストを自動化したツール」です**。<br>
辞書に登録された語彙や説明文は、特定の立場を代表するものではなく、<br>
**今後の社会状況や利用環境の変化に応じて更新・差し替え可能なもの**として設計されています。

## システム構成
本ツールは Google Apps Script（GAS）を用いた簡易チェックシステムです。<br>
投稿前の文面を Google フォームから送信し、以下の流れで処理を行います。
1. Google フォーム<br>
   投稿予定文、投稿予定日を入力
2. Google スプレッドシート<br>
   フォーム送信内容を蓄積
3. JSON 辞書<br>
   誤解を招きやすい語彙と、その注意軸（axes）・補足情報（note）を定義
4. 判定処理（GAS）
   - 語彙の包含チェック
     - 日付依存語彙の場合は日付もチェック
   - 24時間以内の連投チェック（frequencyCheck）
5. 出力<br>
   注意喚起メッセージとして結果をまとめて表示

### 辞書設計
辞書は JSON 形式で管理しています。
- key: 検知対象の語彙
- axes:
  - 1: 特定の日付と結びつくことで政治的・歴史的文脈を帯びる語
  - 2: 近年、侮蔑・揶揄・価値観の押し付けと解釈されやすい語
- dates: axes=1 の場合のみ使用
- note: なぜ注意が必要かの簡易説明

#### 辞書について
本リポジトリに含まれる辞書は、網羅性や正解を目指したものではありません。<br>
本ツールの目的は、「誰もが知っている強い禁忌語」を検出することではなく、<br>
**善意や一般論のつもりで使われがちな語彙が、特定の文脈では誤解を招きうることを可視化すること**にあります。<br>
そのため、以下を意図的に行っています。
- 露骨な差別語・罵倒語の網羅は行わない
- 政治的・倫理的な是非の判断は行わない
- 日本語圏・日本企業のSNS運用という文脈に限定する<br>
辞書の内容は、用途や組織文化に応じて各利用者が自由に編集・拡張することを前提としています。


### 連投判定について
本ツールでは、同一語彙が短時間に繰り返し使用されていないかを確認します。
- Google Apps Script の PropertiesService を利用し、語彙ごとの最終使用時刻を保存しています。
- 24時間以内に再度検知された場合、<br>
  「短時間で繰り返し使用されている可能性があります」
  という注意文を追加します。<br>
これは炎上を「単発の言葉」ではなく、「語彙の積み重ね」として捉えるための簡易的な仕組みです。

## 想定される応用例

本ツールは、SNS広報以外にも以下の用途に応用可能です。

- 社内チャット・メールでの表現チェック
- 異文化コミュニケーションにおける注意喚起
- 教育・研修用途でのケーススタディ<br>

これらの用途に対応するための拡張は、利用者自身による改変を前提としています。

## 現在の制約

本ツールは以下の制約を意図的に受け入れています。

- 文脈理解を行わない
- 正規表現や形態素解析を使用しない
- 語彙の重み付けやスコアリングを行わない<br>

これらは、誤検知や過剰反応を避け、説明可能性を優先するための設計判断です。


## 注意事項
- 本ツールは炎上を防止することを保証するものではありません
- 文脈理解や表現の是非を判断するものではありません
- 最終的な投稿判断は利用者自身に委ねられます
